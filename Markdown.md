# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом необходимо обрамить его звездочкой (*). Пример: *Текст*

Чтобы выделить текст полужирным необходимо обрамить его двумя  звездочками (**). Пример: **Текст**
